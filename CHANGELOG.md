## [1.0.0] - 2026-03-14

### Added
- Initial release
- CLI with commands: list, run, info, progress, reset
- Lab loader with validation and error handling
- Progress tracking system
- Pytest test suite
- GitHub Actions CI/CD

### Labs Added
1. SQL Injection (Beginner)
2. Password Cracking (Beginner)
3. XSS - Cross-Site Scripting (Intermediate)
4. CSRF - Cross-Site Request Forgery (Intermediate)
5. Brute Force (Beginner)
6. Ransomware (Advanced)
7. MITM - Man-in-the-Middle (Advanced)
8. Directory Traversal (Intermediate)
9. Command Injection (Advanced)
10. SSRF - Server-Side Request Forgery (Intermediate)
11. IDOR - Insecure Direct Object Reference (Intermediate)
12. XXE - XML External Entity (Advanced)
13. JWT - JSON Web Token Security (Intermediate)

### Features
- Colorama support for colored output
- Type hints throughout the codebase
- Comprehensive error handling
- Exit codes for CLI commands
- Package installation support (setuptools)
